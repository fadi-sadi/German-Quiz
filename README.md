# German Quiz

Think you know your German? Put it to the test!

**German Quiz** is a trivia game that challenges your German language skills
across CEFR proficiency levels — from beginner (A1) all the way to advanced
(C1). Built with the
[YAML Trivia Engine](https://github.com/fadi-sadi/yaml-trivia-engine).

## How It Works

Pick a level and skill area, then answer as many questions as you can. Each
round shuffles the questions and answer choices, so no two games feel the same.

## Play It

The game is live at
**[fadi-sadi.github.io/German-Quiz](https://fadi-sadi.github.io/German-Quiz/)** —
no install needed, just open and play.

## Run Locally

```bash
npm install
npm run dev
```

Then open <http://localhost:5173> in your browser.
